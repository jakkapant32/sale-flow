ต้องติดตั้ง Web Server (XAMPP/WAMP) เพื่อรัน PHP

วิธีที่ 1: XAMPP
- ดาวน์โหลด: https://www.apachefriends.org/download.html
- ติดตั้ง → เปิด XAMPP Control Panel → Start Apache
- Copy โฟลเดอร์ SalesFlow ไปที่ C:\xampp\htdocs\
- เปิดเบราว์เซอร์: http://localhost/SalesFlow/

วิธีที่ 2: WAMP
- ดาวน์โหลด: https://www.wampserver.com/en/
- ติดตั้ง → เปิด WAMP → รอจนไอคอนเป็นสีเขียว
- Copy โฟลเดอร์ SalesFlow ไปที่ C:\wamp64\www\
- เปิดเบราว์เซอร์: http://localhost/SalesFlow/
